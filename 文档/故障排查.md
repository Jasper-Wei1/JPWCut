# 故障排查

## Windows 安装或 Studio 启动失败

Windows 10/11 x64 用户从仓库根目录双击 `Windows/install.cmd`，不要在 OneDrive、桌面或过深目录中运行。推荐解压到 `D:\JPWClips`。安装脚本会检查 Node.js、PowerShell、Remotion 和本地 Whisper。

Windows ARM 设备暂不在支持范围内，因为本项目的本地 Whisper 安装包使用 x64 二进制。

## `npm run setup` 无法编译 Whisper.cpp

本节只适用于 macOS 或 Linux。

先安装 Apple 命令行开发工具，再重试：

```bash
xcode-select --install
npm run setup
```

## 模型下载中断

重新运行同一个安装命令。完整模型按精确文件大小校验；不完整文件会移入 `工作区/待删除/`，不会被误用。

## 转录提示找不到输入文件

从仓库根目录运行命令，并给中文或带空格的路径加双引号。直播原片应位于 `输入/媒体素材/直播录像/`。

## Studio 找不到直播源

先完成 `npm run clips:review -- prepare ...`。该步骤会将原片链接或复制到 Remotion 的本地生成目录，原片本身不会修改。
