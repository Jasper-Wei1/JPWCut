# 本地语音转录

直播原片经本地 Whisper.cpp 转为带时间码的逐字稿：

```text
直播录像 -> 16 kHz 单声道 WAV -> Whisper.cpp -> 标准化逐字稿 JSON
```

音频不会上传到云端语音服务。直播没有正式口播稿；原声和时间码逐字稿是内容真源。只能依据证据修正明确的错字或专名，听不清的内容必须记录疑点，不得为流畅改写原话。

默认模型是 `small`，约 488 MB。草稿结果放在 `工作区/数据/草稿/`，已确认的审校逐字稿放在 `工作区/数据/已确认/`。

Windows 使用 Whisper.cpp 1.5.5 时不启用 DTW token 级时间戳，并按 128 个 token 分段，规避中文 token 与时间码损坏；macOS 和 Linux 继续使用 token 级时间戳。任何平台参数调整都必须先以代表性短片验证逐字稿质量和时间码范围，再运行整场转录。
