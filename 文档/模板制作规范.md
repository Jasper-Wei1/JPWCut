# 模板制作规范

一个模板必须同时具备以下内容才算完成：

1. 唯一的小写英文模板 ID。该 ID 是机器标识，不翻译。
2. `模板/<id>/模板说明.json` 下的模板说明。
3. `模板/模板索引.json` 中的登记项。
4. 使用共享 Zod 数据结构的 Remotion Composition。
5. `引擎/remotion/public/video-data/` 下的示例数据。
6. `模板/样式画廊/<id>.png` 下由真实模板渲染的代表图片。
7. 面向制作 Skill 的聚焦参考文件。
8. 通过数据、布局、媒体对齐和 TypeScript 测试。

不要只因为效果新奇就登记模板。模板必须帮助观众理解一种会重复出现的信息
结构，并在目标画幅中保持清晰可读。
